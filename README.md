# TaskManager

**TaskManager** is a sleek and intuitive mobile app built with **React Native (Expo)** that helps users organize their daily tasks efficiently. It focuses on simplicity, clarity, and productivity â€” providing users with a clean interface to manage, track, and prioritize their to-dos.

## Features
- **Add, Edit, and Delete Tasks:** Manage tasks effortlessly with smooth animations.
- **Due Dates & Priorities:** Assign dates and categorize by urgency.
- **Mark as Complete:** Instantly toggle task completion status.
- **Persistent Storage:** Data saved locally using **AsyncStorage**.
- **Toast Notifications:** Quick feedback for task actions.
- **Clean UI:** Minimal and responsive layout for distraction-free use.
- **Multi-screen Navigation:** Integrated stack & tab navigation for seamless transitions.

## Tech Stack
- React Native (Expo)
- React Navigation
- AsyncStorage
- Styled Components
- Context API / State Hooks
